聊聊个人任务管理

Posted by Yeoman on 2018-08-03

背景

Q2季度事情非常多而杂，尤其端午那段时间整个人的状态非常混乱。感觉急需一个科学合理的个人任务管理工具。于是在工作节奏缓过来之后，看了一些这方面的文章，然后发现了一个科技类博主JailbreakHum的知乎Live，Live有两期，分成入门和进阶，仔细听完之后比较认同作者看问题看本质的思维方式。

这篇文章主要就是记录下Live的听后感以及自己的一些实践。

Why？

个人任务管理应该是一个私人定制化比较强的事情，但是做这个事情我们要解决的问题应该是一致的。

  1. 会经常被各种琐碎的事情打断，常常理不清任务的优先级
  2. 会经常彻底忘记掉一些不是当下需要马上处理的事情
  3. 脑子里经常惦记着几件重要的事，导致无法安心处理手里的工作
  4. 对于一些生活中的小事养成了一拖再拖的习惯

上面这四点是我列出的我个人的痛点，其中第三点是我觉得最严重的一点。当同时有好几件重要的事情来临的时候，如果没有好好规划任务，就很容易导致效率变得更低。

如果大家没有上述这些痛点的话，那么恭喜你，你的个人任务管理能力已经非常棒了～OR，你的工作还没有对你进行足够的摧残。

关于GTD

谈到任务管理，很多人都会想到GTD（getting things done），但是一去查GTD的内容就会觉得不就是个任务管理么，为什么搞得这么复杂！（摔

Live的作者认为，之所以GTD让人觉得复杂，主要是以下几个原因：

  1. GTD不是单纯的任务管理，是一个本身就很复杂的体系，但是我们在日常任务管理上不需要那么复杂的概念
  2. GTD那本书的内容和现代的任务管理工具严重脱节
  3. 很多工具套用了GTD里面的概念，但是对于相同的功能使用了不同的名字，让使用者困惑

总之，我们首先要明确一点，任务管理不复杂，甚至很简单！

GTD的5个步骤

我们来看下GTD的五个核心步骤：

3.1 收集

  1. 收集任务的时候不要去整理，要批量收集，批量处理。（比如说，我们在写博客的时候，需要收集很多图片，但是我们不想被打断思路，这时候就应该在写文章的时候先打上标记，最后批量收集图片）
  2. 收集任务的时候一定要尽可能多的记录任务的细节，否则一段时间过后就会忘记。
  3. GTD的原则是收集任务的时候如果碰到一些可以马上做掉的小任务尽量马上做掉。

3.2 处理

  1. 确保收集的任务是切实可行的，如果不是，则需要及时删掉。
  2. 如果你的任务只需要几分钟就可以完成，可以搭上标记。（比如可以打上@easy的标签）。然后可以在零碎的时间去完成它们。

3.3 组织

组织是比较重要的一步了，这个关系到后面我们如何去找到我们的要做的任务。

组织任务主要有两个维度：列表和标签。

3.3.1 列表

  1. 列表在各个工具里有不同的名字，比如list，listGroup，project。
  2. 列表可以有层级。

3.4 归档

归档这一步其实是被挺多同学忽略的一步，我觉得归档带来的好处还是显而易见的：

  1. 是对过去一段时间工作的回顾和整理
  2. 在汇报工作的时候可以拿出一份比较完整的报告
  3. 要区分归档和删除，对于没有保留意义的内容应该是删除而不是归档

3.5 执行

关于执行，Live的作者主要给出了下面这几条做事情的tips：

  1. 使用定时器（比如番茄工作法）来记录自己能集中注意力的时间，不断的调整，理想状态是达到1个小时
  2. 尽可能批量地去完成同类的任务
  3. 大部分事情尽量都在当下就去完成，因为过了当下就需要花更多精力去找到当时的那个上下文
  4. 合理拖延，在精神状态不到位的时候，无法处理重要事件，这时候可以先去处理一些简单的任务，减轻焦虑

过滤

讲了这么多，接下来要讲的这个功能才是重头戏，概念非常简单，但却是现代任务管理工具的核心功能。（Live作者这么认为，我也非常认同～）

4.1 为什么过滤很重要

过滤这个功能是传统纸币系统所不能企及的现代任务管理的核心，它可以让我们在任何情境下都可以最快的找到我们当下需要处理的任务。列表和标签本质上都是为了过滤的两个维度。

4.2 三大工具

谈到任务管理工具，不得不提三大任务管理工具：omniFocus，2do，todoist，过滤这个功能在这三个软件里分别叫：透视，智能列表，过滤。（things不在讨论范围，因为things在过滤这一块功能比较弱）

omniFocus

其中omniFocus应该是名气最大的，尤其是以它的复杂而著称。据说这个软件基本没有差评，如果在使用过程中发现不好用，那用户也会觉得是自己不会用这个软件- -（不明觉厉

但是这个作者毫无留情地diss了这个软件- -

2do

2do这个软件功能比较完备，iOS和mac平台都支持，用户体验也比较好。而且是这三者中过滤器语法支持最全的。唯一的缺憾是同步需要通过Dropbox。

这是2do的过滤器语法，可以发现作者确实是把过滤作为非常重要的一个功能来开发的，考虑得非常细。

todoist

但是Live的作者推荐的是todoist这个软件，这个软件符合上面说的所有功能，同步也是通过他们官方的服务器，同时支持api扩展其他服务的调用。但是这个软件是按年收费的。

4.3 my choice

首先，omniFocus肯定不在选择的范畴。虽然作者是极力推荐todoist这个软件，但是todoist比2do擅长的部分我并没有什么需求，而2do已经完全能满足我的日常需求，而且我访问了国内和美国的App Store，2do的评分都比todoist要高。

因此我选择一次性消费，咬咬牙买了2do的license（难道不是因为穷？

one more thing

Live的作者最后有提到一点，信任。我觉得这个词非常得恰当了，我们要足够信任我们的任务管理工具。让他去帮我们找出我们当下应该做的事情，这样我们就可以更专注于我们该做的事情上了。
